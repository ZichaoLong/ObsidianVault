# ObsidianVault 入口

本仓库用于承载从 `~/llm/llm-notes` 迁移、去重、重组后的研究笔记。

当前迁移原则：

- 不迁移 `mhtml` 本体，只把它们作为图片恢复来源。
- 非 `mhtml` 文件都必须进入迁移清单，避免遗漏。
- 正文内容只保留一份；主题页负责承载整理后的正文，来源索引负责保留出处、hash 和迁移状态。
- `review` 与 `defense` 不混入正文，整理为 challenge / defense 链。
